# crunchGamesInternational

HTML5 games, playable for free in a web browser, unblocked (maybe (depends))



**TO CONTRIBUTE GAMES!** Create a fork of this repository and make the changes you want. Then, submit it as a pull request to this repo. It will be manually approved and merged. Be sure to also add the games in their appropriate place in alphabetical order in the list on index.html in the parent folder.
